# Roomora Hotel Booking

Full-stack hotel booking web application built using React + Vite.
Roomora is a full-stack hotel booking platform where users can explore and book a variety of hotel rooms seamlessly. Built using React.js, Node.js, Express, and MongoDB, the platform provides a smooth and responsive booking experience with user authentication, room listings, and booking management.

# 🏨 Roomora - Hotel Booking Platform

Roomora is a full-stack hotel booking web application that allows users to browse hotels, view room details, and book rooms online. The project is built using the MERN stack (MongoDB, Express, React, Node.js).

---

## 🚀 Features

### 👤 User Features
- User registration & login (authentication)
- Browse hotels and room categories
- View detailed room information
- Book available rooms
- View booking history

### 🏨 Hotel Features
- Multiple room types (Deluxe, Standard, Suite, etc.)
- Room availability status
- Pricing display per night
